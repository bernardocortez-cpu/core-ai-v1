# Core AI v1

Core AI v1 is an early prototype of a unified AI platform.

It was built to explore the concept of combining multiple AI models, persistent memory, and intelligent routing inside a single workspace.

## Core ideas
- Multi-model AI experience
- Persistent memory
- Intelligent routing
- Unified AI workspace

## Purpose
This repository represents the foundational prototype of Core AI and is being kept as a portfolio milestone while the next version evolves into a broader AI ecosystem.
